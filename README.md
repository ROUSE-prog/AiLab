# AiLab

AiLab is my personal **AI engineering lab** for building startup-style AI products and experiments.
Live Demo: [AiLab on Vercel](https://your-vercel-url.vercel.app)


The goal of this repository is to explore modern AI product development using the tools and architecture commonly used by AI startups.

# 🚀 Current Project

## Project 01 — Prompt Playground

A streaming prompt playground for experimenting with prompt engineering and model responses.

### Features

* System prompt input
* User prompt input
* Streaming AI responses
* Scrollable output panel
* Clean UI built with Tailwind

### Tech Stack

* Next.js (App Router)
* TypeScript
* Tailwind CSS
* Vercel AI SDK
* OpenAI API

# ⚙️ Running the Project

Clone the repository:

```
git clone https://github.com/YOUR_USERNAME/AiLab.git
```

Navigate to the web app:

```
cd AiLab/apps/web
```

Install dependencies:

```
npm install
```

Start the development server:

```
npm run dev
```

Then open:

```
http://localhost:3000
```

---

# 🔑 Environment Variables

Create a file:

```
apps/web/.env.local
```

Add your OpenAI API key:

```
OPENAI_API_KEY=your_api_key_here
```
